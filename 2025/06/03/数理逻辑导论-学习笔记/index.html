Equivalence Relation 等价关系:

  • Reflexive (自反): ∀x∈A\forall x \in A∀x∈A, xRxxRxxRx.

  • Sysmmetric (对称): ∀x∈A\forall x \in A∀x∈A, if xRyxRyxRy, then yRxyRxyRx.

  • Transitive (传递): ∀x,y,z∈A\forall x, y, z \in A∀x,y,z∈A, if xRyxRyxRy and yRzyRzyRz, then xRzxRzxRz.

  • A binary relation RRR on a nonempty set AAA is an equivalence relation if and only if RRR is reflexive, sysmmetric and transitive.

Equivalence Class 等价类:

  • [x]R={y∈A∣xRy}[x]_R = \{y \in A \mid xRy\}[x]R​={y∈A∣xRy} is the set of all elements of AAA that are equivalent to xxx. (RRR is an equivalence relation over a set AAA)

  • Every element belongs to exactly one equivalence class.

Partial Order Relation 偏序关系:

  • Antisymmetric (反对称): ∀x,y∈A\forall x, y \in A∀x,y∈A, if xRyxRyxRy and yRxyRxyRx, then x=yx = yx=y.

  • A binary relation RRR on a nonempty set AAA is a partial order if and only if RRR is reflexive, antisysmmetric and transitive.

  • Minimal Element: For x∈Ax \in Ax∈A, no y∈Ay \in Ay∈A that satisfys yRxyRxyRx exists.

  • Maximal Element: For x∈Ax \in Ax∈A, no y∈Ay \in Ay∈A that satisfys xRyxRyxRy exists.

Total Order Relation 全序关系 / Linear Order Relation:

  • A partial order relation RRR on a set AAA is a total order (linear order), iff ∀x,y∈A\forall x, y \in A∀x,y∈A, either xRyxRyxRy or yRxyRxyRx.

  • Intuitively, ant two elements are comparable.

Injective, Surjective & Bijective:

  • Injective (one-to-one, 单射): Each element of the codomain is mapped to by at most one element of the domain.

  • Surjective (onto, 满射): Each element of the codomain is mapped to by at least one element of the domain.

  • Bijective (one-to-one correspondence, 双射): Each element of the codomain is mapped to by exactly one element of the domain. (both injective and surjective)

Proof by Induction 归纳证明:

  • Base Case: We need to show that P(n)P(n)P(n) is true for the smallest possible value of nnn. e.g., P(n0)P(n_0)P(n0​) is true.

  • Inductive Step: Show that the statement P(k+1)P(k + 1)P(k+1) is true.

Propositions & Connectives & Language in General

Proposition 命题:

  • Proposition: A proposition is a declarative sentence that can be judged as either true or false.

    e.g., 2+2=52 + 2 = 52+2=5 is a proposition, while x+y<0x + y < 0x+y<0 is not a proposition.

  • Atomic Proposition (原子命题): A proposition that does not contain any smaller part that is still a proposition is called an atomic proposition.

  • Compound Proposition (复合命题): A proposition that involves the assembly of multiple propositions is called a compound proposition.

Well-Formed Formulas (WFF):

  • Every atom is a WFF.

  • If α\alphaα is a WFF, then so is (¬α)(\neg \alpha)(¬α).

  • If α\alphaα and β\betaβ are WFF, then so are (α∧β),(α∨β),(α→β),(α↔β)(\alpha \land \beta), (\alpha \lor \beta), (\alpha \to \beta), (\alpha \leftrightarrow \beta)(α∧β),(α∨β),(α→β),(α↔β).

Some Definitions:

Common Properties of WFF:

  • WFFs in LP\mathscr{L}^PLP are non-empty expressions.

  • Every WFF in LP\mathscr{L}^PLP has an equal number of opening and closing brackets.

  • Every proper prefix of WFF in LP\mathscr{L}^PLP has more opening brackets than closing brackets. Every proper suffix of WFF in LP\mathscr{L}^PLP has more closing brackets than opening brackets.

    Hence, proper prefix and proper suffix are not WFF in LP\mathscr{L}^PLP.

    • Leaves are atoms.

  • WFF ↔\leftrightarrow↔ a parse tree of it.

  • Subformula: If GGG occurs within FFF, then GGG is a subformula of FFF.

    Proper Subformula: G≠FG \ne FG=F.

  • Immediate Subformula: The children of a formula in its parse tree.

    Leading Connective: The connective that is used to join children.

Unique Readability Theorem:

  • There is a unique way to construct every WFF.

Simplification:

  • Convention (公约): We may drop the outermost parentheses, i.e., FFF is the same as (F)(F)(F).

    • First group the rightmost occurrence. e.g., p→q→rp \to q \to rp→q→r means p→(q→r)p \to (q \to r)p→(q→r).

  • Definition: Two formulas AAA and BBB are logically equivalent iff they have the same value under any valuation.

Logical Identities:

  • Identity Laws (恒等率): p∧T≡pp \land T \equiv pp∧T≡p, p∨F≡pp \lor F \equiv pp∨F≡p.

  • Domination Laws (支配律): p∧F≡Fp \land F \equiv Fp∧F≡F, p∨T≡Tp \lor T \equiv Tp∨T≡T.

  • Idempotent Laws (幂等率) : p∧p≡pp \land p \equiv pp∧p≡p, p∨p≡pp \lor p \equiv pp∨p≡p.

  • Double Negation Laws (双非率): ¬(¬p)≡p\neg (\neg p) \equiv p¬(¬p)≡p.

  • Commutative Laws (交换律)

  • Associative Laws (结合律): (p∧q)∧r≡p∧(q∧r)(p \land q) \land r \equiv p \land (q \land r)(p∧q)∧r≡p∧(q∧r), (p∨q)∨r≡p∨(q∨r)(p \lor q) \lor r \equiv p \lor (q \lor r)(p∨q)∨r≡p∨(q∨r).

  • Distributive Laws (分配率): p∨(q∧r)≡(p∨q)∧(p∨r)p \lor (q \land r) \equiv (p \lor q) \land (p \lor r)p∨(q∧r)≡(p∨q)∧(p∨r), p∧(q∨r)≡(p∧q)∨(p∧r)p \land (q \lor r) \equiv (p \land q) \lor (p \land r)p∧(q∨r)≡(p∧q)∨(p∧r).

  • De Morgan’s Laws (德摩根率): ¬(p∧q)≡¬p∨¬q\neg (p \land q) \equiv \neg p \lor \neg q¬(p∧q)≡¬p∨¬q, ¬(p∨q)≡¬p∧¬q\neg (p \lor q) \equiv \neg p \land \neg q¬(p∨q)≡¬p∧¬q.

  • Absorption Laws (吸收率): p∨(p∧q)≡pp \lor (p \land q) \equiv pp∨(p∧q)≡p, p∧(p∨q)≡pp \land (p \lor q) \equiv pp∧(p∨q)≡p.

  • Negation Laws (否定率): p∧¬p≡Fp \land \neg p \equiv Fp∧¬p≡F, p∨¬p≡Tp \lor \neg p \equiv Tp∨¬p≡T.

  • Implication: p→q≡¬p∨qp \to q \equiv \neg p \lor qp→q≡¬p∨q.

  • Contrapositive (逆否): p→q≡¬q→¬pp \to q \equiv \neg q \to \neg pp→q≡¬q→¬p.

  • Equivalence: p↔q≡(p→q)∧(q→p)p \leftrightarrow q \equiv (p \to q) \land (q \to p)p↔q≡(p→q)∧(q→p).

  • Redundancy (累赘): →\to→ is said to be definable in terms of ¬\neg¬ and ∨\lor∨, because (p→q)≡((¬p)∨q)(p \to q) \equiv ((\neg p) \lor q)(p→q)≡((¬p)∨q).

Resolution Proof System

aka. Refutation (反驳) System.

Normal Form:

  • This is a standardized representation of logical formulas.

  • Literal (单式, 文字): An atomic formula of the negation of that.

  • Conjunctive Clause (合取子句): A conjuction of literals.

    Disjunctive Clause (析取子句): A disjuction of literals.

  • Conjunctive Normal Form (CNF, 合取范式): A conjuction of disjunctive clauses.

    (a∨⋯∨b)∧⋯∧(c∨⋯∨d)(a \lor \cdots \lor b) \land \cdots \land (c \lor \cdots \lor d) (a∨⋯∨b)∧⋯∧(c∨⋯∨d)

    Disjunctive Normal Form (DNF, 析取范式): A disjunction of conjunctive clauses.

    (a∧⋯∧b)∨⋯∨(c∧⋯∧d)(a \land \cdots \land b) \lor \cdots \lor (c \land \cdots \land d) (a∧⋯∧b)∨⋯∨(c∧⋯∧d)

Properties:

  • If AAA is a formula in CNF, then ¬A\neg A¬A is logically equivalent to a formula in DNF.

    If AAA is a formula in DNF, then ¬A\neg A¬A is logically equivalent to a formula in CNF.

  • Every formula A∈Form(LP)A \in \text{Form}\left(\mathscr{L}^P\right)A∈Form(LP) is logically equivalent to some formula in CNF and DNF.

  • Principal CNF / DNF: Each clause in a principal CNF / DNF contains all propositional variables exactly once, and no two clauses are the same. e.g., (p∨¬q∨r)∧(p∨q∨¬r)(p \lor \neg q \lor r) \land (p \lor q \lor \neg r)(p∨¬q∨r)∧(p∨q∨¬r).

Resolution proof system applies only to formulas in CNF. It is used to prove contradictions (⊥\bot⊥).

Inference Rule:

  • Resolution Inference Rule:

    (α∨p)((¬p)∨β)(α∨β)\frac{(\alpha \lor p) \quad ((\neg p) \lor \beta)}{(\alpha \lor \beta)} (α∨β)(α∨p)((¬p)∨β)​

    Each step of the proof produces one clause (Resolvent, 消解子句) from two previous clauses by applying the resolution rule.

    Special cases:

    • Unit Resolution: (α∨p)(¬p)α\dfrac{(\alpha \lor p) \quad (\neg p)}{\alpha}α(α∨p)(¬p)​

    • Contradiction: p(¬p)⊥\dfrac{p \quad (\neg p)}{\bot}⊥p(¬p)​

Proof Procedure:

  • To prove Σ⊢φ\Sigma \vdash \varphiΣ⊢φ, we prove Σ∪{¬φ}⊢⊥\Sigma \cup \{\neg \varphi\} \vdash \botΣ∪{¬φ}⊢⊥ instead.

  • First, convert each formula in Σ\SigmaΣ and ¬φ\neg \varphi¬φ to CNF.

  • Next, split CNF formulas at the ∧\land∧s.

    • A CNF can be split to a set of disjunctive clauses. e.g., (p∨q)∧(q∨¬r)∧s(p \lor q) \land (q \lor \neg r) \land s(p∨q)∧(q∨¬r)∧s can be split to {p,q},{q,¬r},{s}\{p, q\}, \{q, \neg r\}, \{s\}{p,q},{q,¬r},{s}.

    • We treat ⊥\bot⊥ as a clause containing no literal, i.e., the empty set ∅\varnothing∅. Since it contains no true literal, it is false.

  • Then, keep applying the resolution inference rule until either:

    • we have the empty clause ⊥\bot⊥, which means we have proved that Σ⊢φ\Sigma \vdash \varphiΣ⊢φ, or

    • the rule can no longer be appiled to give a new formula, which means φ\varphiφ cannot be proven from Σ\SigmaΣ.

Soundness & Completeness:

  • The resolution proof system is sound and complete.

Process of program verification:

Formal Specification

Two important components of a specification:

  • Precondition: The state before the program executes.

  • Postcondition: The state after the program executes.

Our formal notation:

  • is based on FOL logic.

  • applies to imperative programs.

  • uses Hoare Triples.

  • A Hoare triple (∣P∣) C (∣Q∣)(|P|) ~ C ~ (|Q|)(∣P∣) C (∣Q∣):

    • Code to be verified: CCC

  • Hoare triple (∣P∣) C (∣Q∣)(|P|) ~ C ~ (|Q|)(∣P∣) C (∣Q∣) is satisfied under partial correctness iff

    • for every state s1s_1s1​ that satisfies PPP,
    • if execution of CCC starting from state s1s_1s1​ terminates in a state s2s_2s2​,
    • then state s2s_2s2​ satisfies condition QQQ.
    • (if execution of CCC does not terminate, it’s ok)

    which is denoted as

    ⊨par(∣P∣) C (∣Q∣)\models_{\text{par}} (|P|) ~ C ~ (|Q|) ⊨par​(∣P∣) C (∣Q∣)

Total Correctness:

  • Hoare triple (∣P∣) C (∣Q∣)(|P|) ~ C ~ (|Q|)(∣P∣) C (∣Q∣) is satisfied under total correctness iff

    • for every state s1s_1s1​ that satisfies PPP,
    • execution of CCC starting from state s1s_1s1​ terminates in a state s2s_2s2​,
    • then state s2s_2s2​ satisfies condition QQQ.
    • (execution of CCC must terminate)

    which is denoted as

    ⊨tot(∣P∣) C (∣Q∣)\models_{\text{tot}} (|P|) ~ C ~ (|Q|) ⊨tot​(∣P∣) C (∣Q∣)

Hoare Logic

Presentation of a Proof:

Axioms:

  • Assignment:

    (∣Q[E/x]∣) x=E (∣Q∣)\frac{}{(|Q[E / x]|) ~ x = E ~ (|Q|)} (∣Q[E/x]∣) x=E (∣Q∣)​

  • Implied Rule: Precondition Strengthening:

    P→P′(∣P′∣) C (∣Q∣)(∣P∣) C (∣Q∣)\frac{P \to P' \quad (|P'|) ~ C ~ (|Q|)}{(|P|) ~ C ~ (|Q|)} (∣P∣) C (∣Q∣)P→P′(∣P′∣) C (∣Q∣)​

  • Implied Rule: Postcondition Weakening:

    (∣P∣) C (∣Q′∣)Q′→Q(∣P∣) C (∣Q∣)\frac{(|P|) ~ C ~ (|Q'|) \quad Q' \to Q}{(|P|) ~ C ~ (|Q|)} (∣P∣) C (∣Q∣)(∣P∣) C (∣Q′∣)Q′→Q​

  • Composition:

    (∣P∣) C1 (∣Q∣)(∣Q∣) C2 (∣R∣)(∣P∣) C1;C2 (∣R∣)\frac{(|P|) ~ C_1 ~ (|Q|) \quad (|Q|) ~ C_2 ~ (|R|)}{(|P|) ~ C_1; C_2 ~ (|R|)} (∣P∣) C1​;C2​ (∣R∣)(∣P∣) C1​ (∣Q∣)(∣Q∣) C2​ (∣R∣)​

  • If Statements:

    (∣P∧B∣) C1 (∣Q∣)(∣P∧¬B∣) C2 (∣Q∣)(∣P∣) if B {C1} else {C2} (∣Q∣)\frac{(|P \land B|) ~ C_1 ~ (|Q|) \quad (|P \land \neg B|) ~ C_2 ~ (|Q|)}{(|P|) ~ \text{if } B ~ \{C_1\} \text{ else } \{C_2\} ~ (|Q|)} (∣P∣) if B {C1​} else {C2​} (∣Q∣)(∣P∧B∣) C1​ (∣Q∣)(∣P∧¬B∣) C2​ (∣Q∣)​

  • While Statements / Partial-While: (do not yet require termination)

    (∣I∧B∣) C (∣I∣)(∣I∣) while B {C} (∣I∧¬B∣)\frac{(|I \land B|) ~ C ~ (|I|)}{(|I|) ~ \text{while } B ~ \{C\} ~ (|I \land \neg B|)} (∣I∣) while B {C} (∣I∧¬B∣)(∣I∧B∣) C (∣I∣)​

    where III is a loop invariant which holds both before we execute CCC and after CCC terminates.

    To prove termination (total correcness), we need to find a variant, which is an integer expression that:

    • always stays non-negative, and
    • strictly decrease in every loop iteration.

    The automatic extraction of useful variants or termination expressions cannot be realized.

Decidability

Decision Problem:

  • The answer is yes or no.

Decidable:

  • If there exists an algorithm that always halts (停止) with a yes / no answer for every input.

e.g.

  • Collatz Conjecture: Algorithmically undecidable.
  • The Halting Problem: Undecidable.
  • The Partial Correctness Problem: Undecidable.
